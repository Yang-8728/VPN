# Outline VPN 快速部署指南

> 在泰国接收中国 App 验证码的最简单方案

## 项目简介

这是一个极简的 VPN 部署指南，帮助在泰国的用户快速搭建中国 VPN 服务器，用于接收中国 App（微信、支付宝、淘宝等）的短信验证码。

### 核心特点

- **极简部署**：一条命令搞定，5 分钟完成
- **超低成本**：使用 1 小时约 0.1-0.2 元，用完即删
- **完全免费**：使用开源的 Outline，服务端和客户端都免费
- **一次性使用**：接收完验证码立即删除，不留痕迹

### 技术方案

- **服务器**：华为云中国区最小配置（1核1GB）
- **VPN 软件**：Outline（Google Jigsaw 开源项目）
- **客户端**：Outline for iOS（App Store 免费下载）
- **协议**：Shadowsocks（安全加密）

## 快速开始

### 第一步：购买华为云服务器

1. 登录 [华为云控制台](https://www.huaweicloud.com/)
2. 购买弹性云服务器 ECS
   - 区域：中国大陆（北京/上海/广州）
   - 配置：1核1GB（最小配置）
   - 镜像：Ubuntu 20.04 或 22.04
   - 计费：按需计费
3. 配置安全组：开放所有 TCP 和 UDP 端口
4. 记下服务器的公网 IP

**详细步骤** 👉 查看 [使用说明.md](./使用说明.md) 第一步

### 第二步：运行部署脚本

SSH 连接到服务器后，运行：

```bash
# 下载脚本
wget https://raw.githubusercontent.com/你的用户名/你的仓库/main/deploy-outline.sh

# 运行脚本
sudo bash deploy-outline.sh
```

脚本会自动：
- 安装 Docker
- 部署 Outline Server
- 生成访问密钥（ss:// 开头的字符串）

**⚠️ 重要：保存输出的访问密钥！**

### 第三步：iPhone 连接使用

1. 在 App Store 下载 **Outline** 客户端（免费）
2. 打开 Outline，点击 **+** 添加服务器
3. 粘贴访问密钥（ss:// 开头的字符串）
4. 点击连接

### 第四步：验证

1. 连接后访问 https://ip.sb
2. 应该显示中国的 IP 地址
3. 打开中国 App，接收验证码

### 第五步：用完删除

1. 在 Outline 中断开连接
2. 登录华为云控制台删除服务器
3. 计费立即停止

## 注意事项

1. **必须选择中国区域**：服务器必须在中国大陆，不能选香港或海外
2. **安全组配置**：必须开放端口，否则无法连接
3. **用完立即删除**：按需计费会持续扣费，用完立即删除服务器
4. **保存访问密钥**：部署脚本只运行一次，访问密钥要保存好
